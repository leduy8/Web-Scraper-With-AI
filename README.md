# Web Scraper with AI

Scrape websites and parse the DOM content with LLM.

## Features

- Scrape websites by URL
- Parse the DOM content with LLM

## Tech

- Python
- Streamlit
- Any LLM (Example: Ollama)
- Selenium
- BeautifulSoup

## Installation

```sh
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
streamlit run main.py
```

## Usage

```sh
streamlit run main.py
```
